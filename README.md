# physics-tools
# الموقع الرسمي للأستاذ محمد قادري

**التجارب الفيزيائية باستخدام الذكاء الصناعي**

هذا المشروع عبارة عن موقع تعليمي يهدف إلى عرض تجارب فيزيائية تفاعلية باستخدام تقنيات الذكاء الصناعي.  
الموقع مصمم ليكون مرجعًا للطلاب والمعلمين لفهم الفيزياء بطريقة ممتعة وعملية.

---

## 📂 هيكل المشروع
physics-tools │ ├── index.html           # الصفحة الرئيسية ├── css/style.css        # تنسيقات الموقع ├── js/script.js         # الأكواد البرمجية ├── images/              # صور وشعارات الموقع ├── data/experiments.json # قاعدة بيانات التجارب └── pages/ ├── experiments.html # صفحة التجارب ├── lessons.html     # صفحة الدروس └── contact.html     # صفحة التواصل
---

## 🎯 أهداف المشروع

- عرض تجارب فيزيائية تعليمية بطريقة تفاعلية.
- تسهيل فهم المفاهيم الفيزيائية للطلاب.
- دمج الذكاء الصناعي في التعليم العملي.
- توفير محتوى عربي سهل القراءة ومتوافق مع جميع الأجهزة.
